# RobUS
Code pour robot RobUS de l'équipe 100ID

# Compilation
```
#define ROBUS_A 0
#define ROBUS_B 1

#define __ROBUS__ ROBUS_A
```
Pour compiler sur le robot désiré changer la valeur de `__ROBUS__` \
Robus A: `#define __ROBUS__ ROBUS_A` \
Robus B: `#define __ROBUS__ ROBUS_B`
